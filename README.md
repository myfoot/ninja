# Ninja

the library behind the scenes

## Installation

Add this line to your application's Gemfile:

    gem 'ninja'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install ninja

## Usage

### Hash

#### accessor 
```ruby
h = {hoge: 1, foo: 2}
h.hoge # => 1
h.foo # => 2
```
